What Was Done

🤖 AI Opponent (Seeker) — DONE

Implemented a complete AI system for the Seeker role with 3 difficulty levels. The AI uses path simulation to predict Hider movement and selects optimal launch angles/powers.

  • Easy: ±15% aim error, 70% power, 0 bounce simulation
  • Medium: ±8% aim error, 85% power, 1 bounce simulation
  • Hard: ±3% aim error, 95% power, 2 bounces, seeks power-up orbs
  • Tracks last-known Hider position via sonar pings and line-of-sight
  • Visual thinking indicator (pulsing dots above Seeker ball)
  • Double-fling guard prevents accidental duplicate launches
  • Integrated into MainMenu with CPU toggle and difficulty selector

Files: src/components/GameCanvas.tsx (~130 new lines for AI), src/components/MainMenu.tsx
